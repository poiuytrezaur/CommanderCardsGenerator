Comment utiliser l'éditeur

Barre d'outils

  • Charger JSON — Importe un fichier .json contenant un tableau de decks. Remplace les données en cache.
  • Télécharger JSON — Exporte tous les decks dans un fichier decks.json.
  • Exporter HTML — Génère un fichier HTML statique prêt à imprimer (format A4, cartes 63 × 88 mm).
  • JSON — Bascule vers l'éditeur JSON brut pour le deck sélectionné. Utile pour coller des données ou corriger manuellement.
  • Vider le cache — Efface les données sauvegardées dans le navigateur (irréversible).

Gestion des decks

  • Cliquez sur + dans la colonne de gauche pour créer un nouveau deck vierge.
  • Cliquez sur un deck dans la liste pour le sélectionner et l'éditer.
  • Le bouton Supprimer ce deck en bas du formulaire supprime définitivement le deck actif.
  • Les modifications sont sauvegardées automatiquement dans le cache du navigateur à chaque changement.

Champs du formulaire

  • Commander (EN) — Nom anglais exact du commander. L'illustration est récupérée automatiquement depuis Scryfall.
  • Nom personnalisé — Nom affiché sur la fiche (peut être en français).
  • Couleurs — Cliquez sur les ronds W / U / B / R / G pour sélectionner l'identité de couleur.
  • Image Y — Position verticale du recadrage de l'art : top, center, 70%…
  • Fond — Dégradé CSS du corps de la carte. Cliquez ⟳ Auto pour le générer depuis la couleur Accent.
  • Stratégie — Description courte du style de jeu (1-3 phrases).
  • Mots-clés — Mécaniques principales séparées par des virgules.
  • Lignes de victoire — Ajoutez les combos ou séquences gagnantes. Chacune apparaît en puce sur la fiche.
  • Courbe de mana — Nombre de cartes par CMC. Cliquez + pour ajouter une colonne.
  • Mana moyen — CMC moyen affiché sous le graphique.

Apparence de la fiche

  • Chaque champ de couleur dispose d'un sélecteur visuel et d'une case hexadécimale synchronisés.
  • L'aperçu à droite (ou flottant sur mobile) se met à jour en temps réel.
  • Les icônes ? à côté des labels affichent des infobulles d'aide au survol.

Astuce raccourci
